# AGENTS.md

This repository's authoritative agent guide is **`CLAUDE.md`** at the
repository root. Every agent runtime that reads `AGENTS.md` should treat
that file as the canonical source.

`AGENTS.md` previously described an older "GSD" framework that has been
superseded by Fenix (`/fenix-auto`). See `CLAUDE.md` for the current
workflow, agent roster, stack, conventions, and what was explicitly
removed from the prior framework.
